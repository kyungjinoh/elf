# NGL App UI

A Vite + React + Tailwind CSS implementation of the NGL app homepage.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Add profile image:
   - Place your profile image at `public/profile.png`
   - The component will use a placeholder if the image is not found

3. Run the development server:
```bash
npm run dev
```

## Features

- Mobile-first responsive design
- Gradient cards with smooth transitions
- Interactive copy link functionality
- Modern UI matching the NGL app design
